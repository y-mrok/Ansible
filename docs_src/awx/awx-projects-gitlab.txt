GitLab のプロジェクトのアドレスを取得
**************************************************
#. 実行するプレイブックを含むプロジェクトをクリック

   .. image:: img/2021-07-25_14h22_33.png

#. :guilabel:`Clone` をクリック

   .. image:: img/2021-07-25_14h27_11.png

#. 「 Clone with SSH 」欄の URL をクリップボードにコピー

   .. image:: img/2021-07-25_14h27_31.png

#. クリップボードにコピーした URL をテキストエディタに貼り付け

   .. image:: img/2021-07-25_14h23_50.png
