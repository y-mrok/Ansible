##################################################
管理対象ノードに接続する準備
##################################################
管理対象ノードにコントロールノードから接続するときのアカウントを作成します。今回使用する管理対象ノードの OS は CentOS 8.3 (2011) です。

.. note::

   コントロールノードと管理対象ノードは SSH 接続します。管理対象ノードに openssh-server がインストール済みであり、設定がなされていなければなりません。
