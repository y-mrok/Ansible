**************************************************
論理演算子
**************************************************
- 条件内に複数種類の論理演算子が含まれる場合、定められた優先順位に従って条件を評価する
- 条件をカッコ ``(`` ``)`` でくくって、明示的に評価順序を変更できる

.. list-table::
   :header-rows: 1
   :widths: 2, 3, 1

   * - 論理演算子
     - true の状態
     - 優先順位
   * - 条件X and 条件Y
     - 条件X と条件Y がともに True のとき
     - 2
   * - 条件X or 条件Y
     - 条件X か条件Y のどちらかが True のとき
     - 3
   * - not 条件X
     - 条件X が False のとき
     - 1
