**************************************************
比較演算子
**************************************************
.. list-table::
   :header-rows: 1
   :widths: 1, 7

   * - 比較演算子
     - true の状態
   * - X == Y
     - X の値と Y の値が等しいとき
   * - X != Y
     - X の値と Y の値が等しくないとき
   * - X > Y
     - X の値が Y の値より大きいとき
   * - X >= Y
     - X の値が Y の値より大きいか等しいとき
   * - X < Y
     - X の値が Y の値より小さいとき
   * - X <= Y
     - X の値と Y の値より小さいか等しいとき
