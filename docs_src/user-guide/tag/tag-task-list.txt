**************************************************
タスクごとに設定したタグを確認
**************************************************
``ansible-playbook`` コマンドに ``--list-tasks`` パラメーターを指定してプレイを実行すると、実行対象のタスクの一覧を表示します。このとき、タスクに設定したタグを併せて表示します。このパラメーターはタスクの一覧を作成するだけで、タスクを実行しません。

「 :ref:`tag-task-tags` 」のプレイに ``--list-tasks`` パラメーターを指定して確認した結果です。
