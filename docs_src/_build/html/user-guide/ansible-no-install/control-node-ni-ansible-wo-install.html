コントロールノードに Ansible をインストール¶

Ubuntu Desktop 20.04.2 LTS のコントロールノードに Ansible をインストールします。

$ sudo apt install -y python3-pip
$ pip3 install ansible
$ pip3 show ansible
$ pip3 show ansible-base
$ ansible --version

上記のコマンドを実行したログです。
