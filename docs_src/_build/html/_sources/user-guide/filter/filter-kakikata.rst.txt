##################################################
フィルターの書き方
##################################################
変数と適用するフィルターは ``|`` で繋げて記述します。

.. code-block:: yaml

   変数 | フィルター

左側に指定した変数に設定した値を右側に設定したフイルターで加工（変換）します。変数に設定した値自体は変更されません。

複数のフィルターを ``|`` で繋げて適用できます。

.. code-block:: yaml

   変数 | フィルター1 | フイルター2
