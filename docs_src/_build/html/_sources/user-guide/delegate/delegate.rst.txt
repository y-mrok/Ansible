.. _delegate:

##################################################
実行を委任
##################################################
- ``delegate_to`` ディレクティブを使用してタスクの実行を任意のノードに委任する
- 委任先の管理対象ノードはインベントリに記載していなくても良い
-  委任先はインベントリ内のホスト名、 IP アドレスやホスト名（名前解決ができる場合）で指定する
