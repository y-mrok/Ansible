Slack に通知¶

AWX でジョブの終了時などに Slack に通知します。

Note

通知先は Slack 以外のアプリケーションも指定できます。

Slack の設定¶

Slack で Bot User を設定します。

  1. Slack API (https://api.slack.com/) のページを開く

  2. Create an app をクリック

  3. From scratch をクリック

    Note

    下図が表示された場合、 Create an APP をクリックすると上図が表示されます。

  4. 次の値を入力

    項目

    入力 / 選択する値

    App Name

    アプリケーション名

    Pick a workspace to develop your app in:

    プルダウンリストからアプリケーションを追加するワークスペースを選択

  5. Create App をクリック

  6. Bots をクリック

  7. Review Scopes to add をクリック

  8. Add an OAuth Scope をクリックして次の 3 つのスコープを選択

    • chat:write

    • chat:write.public

    • incoming-webhock

    ↓

  9. Install to Workspace をクリック

  10. Test APP の東欧先はどちらにしますか？ のプルダウンリストからメッセージの投稿先のチャンネルを選択

  11. 許可する をクリック

  12. copy をクリック（トークンをクリップボードにコピー）

  13. コピーしたトークンをテキストエディターに貼り付け

  14. Slack で ＋ アプリを追加する をクリック

  15. “Test App” をクリック

  16. “Test App” が追加されたことを確認

通知設定¶

  1. 通知 をクリック

  2. 追加 をクリック

  3. 次の値を入力

    項目

    入力 / 選択する値

    名前

    通知の名前

    説明

    通知の説明

    組織

    「 組織 : Organizations 」で登録した組織を選択

    タイプ

    プルダウンリストから Slack を選択

    送信先チャンネル

    メッセージの送信先のチャンネル

    トークン

    テキストエディターに貼り付けたトークンを貼り付け

  4. 保存 をクリック

  5. 通知に戻る をクリック

  6. 作成した通知の「アクション」 列のベルマークをクリック

  7. ステータス列の Successful を確認

Note

複数のチャンネルを通知先に指定できます。

テンプレートに通知を設定¶

通知を設定¶

  1. テンプレート をクリック

  2. 通知を設定するテンプレートをクリック

  3. 通知 をクリック

  4. 通知する種類をオン（今回はすべてオンにしました）
