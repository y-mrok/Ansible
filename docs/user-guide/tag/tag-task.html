タスクにタグをつける¶

  • tags ディレクティブを使用してタスクにタグをつける

  • 1 つのタスクに複数のタグを設定できる

タグをつける¶

tags ディレクティブを使用してタスクにタグを付けます。

---
- name: Examples of tag usage
  hosts: all
  gather_facts: no

  tasks:
    - name: task#1
      ansible.builtin.debug:
      tags: job1
    - name: task#2
      ansible.builtin.debug:
      tags:
        - job2
    - name: task#3
      ansible.builtin.debug:
      tags: job1, job2
    - name: task#4
      ansible.builtin.debug:
      tags:
        - job1
        - job3
    - name: task#5
      ansible.builtin.debug:
    - name: task#6
      ansible.builtin.debug:
      tags: job3
    - name: task#7
      ansible.builtin.debug:
      tags:
        - job2
        - job3

タスクにタグを 1 つだけ付けるときは tags ディレクティブに続けてタグを指定するか、シーケンスでタグを指定します。

- name: task#1
  ansible.builtin.debug:
  tags: job1

- name: task#2
  ansible.builtin.debug:
  tags:
    - job2

タスクに複数のタグを付けるときは tags ディレクティブに続けてカンマ , で区切ってタグを指定するか、シーケンスでタグを指定します。

- name: task#3
  ansible.builtin.debug:
  tags: job1, job2

- name: task#4
  ansible.builtin.debug:
  tags:
    - job1
    - job3

すべてのタスクにタグを設定する必要はありません。タスクにタグが不要なら設定しません。

- name: task#5
  ansible.builtin.debug:

タスクごとに設定したタグを確認¶

ansible-playbook コマンドに --list-tasks パラメーターを指定してプレイを実行すると、実行対象のタスクの一覧を表示します。このとき、タスクに設定したタグを併せて表示します。このパラメーターはタスクの一覧を作成するだけで、タスクを実行しません。

「 タグをつける 」のプレイに --list-tasks パラメーターを指定して確認した結果です。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml --list-tasks

playbook: tag1.yml

  play #1 (all): Examples of tag usage  TAGS: []
    tasks:
      task#1    TAGS: [job1]
      task#2    TAGS: [job2]
      task#3    TAGS: [job1, job2]
      task#4    TAGS: [job1, job3]
      task#5    TAGS: []
      task#6    TAGS: [job3]
      task#7    TAGS: [job2, job3]
piyo@vmpc:~/example/tag$

指定したタグを設定したタスクだけを実行¶

ansible-playbook コマンドの -t パラメーターまたは --tags パラメーターでタグを指定してプレイを実行すると、指定したタグを設定したタスクだけを実行します。複数のタグを指定するときはカンマ , でタグを区切って列記します。

「 タグをつける 」のプレイにタグを指定せず実行したログです。すべてのタスクを実行しました。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml

PLAY [Examples of tag usage] ****************************************************************************************************************

TASK [task#1] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#2] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#3] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#4] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#5] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#6] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#7] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

PLAY RECAP **********************************************************************************************************************************
hydrogen                   : ok=7    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0   

piyo@vmpc:~/example/tag$

タグ job1 を設定したタスクを実行したログです。タグ job1 を設定したタスクだけを実行しました。複数のタグを設定したタスクはいずれかのタグが対象であれば実行します。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml -t job1

PLAY [Examples of tag usage] ****************************************************************************************************************

TASK [task#1] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#3] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#4] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

PLAY RECAP **********************************************************************************************************************************
hydrogen                   : ok=3    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0   

piyo@vmpc:~/example/tag$

--list-tasks パラメーターを併用して、実行対象のタスクを確認したログです。上述のログで実行したタスクだけを表示します。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml -t job1 --list-tasks

playbook: tag1.yml

  play #1 (all): Examples of tag usage  TAGS: []
    tasks:
      task#1    TAGS: [job1]
      task#3    TAGS: [job1, job2]
      task#4    TAGS: [job1, job3]
piyo@vmpc:~/example/tag$

指定したタグを設定したタスクの実行をスキップ¶

ansible-playbook コマンドに --skip-tags パラメーターでタグを指定してプレイを実行すると、指定したタグを設定したタスクをスキップして実行します。複数のタグを指定するときはカンマ , でタグを区切って列記します。

「 タグをつける 」のプレイにタグ job1 を設定したタスクを実行したログです。タグ job1 を設定していない、または、タグを設定していないタスクを実行しました。複数のタグを設定したタスクはいずれかのタグが対象であればスキップします。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml --skip-tags job1

PLAY [Examples of tag usage] ****************************************************************************************************************

TASK [task#2] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#5] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#6] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#7] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

PLAY RECAP **********************************************************************************************************************************
hydrogen                   : ok=4    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0   

piyo@vmpc:~/example/tag$

--list-tasks パラメーターを併用して、実行対象のタスクを確認したログです。 --skip-tags パラメーターで指定したタグを設定したタスクを除いて表示します。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml --skip-tags job1 --list-tasks

playbook: tag1.yml

  play #1 (all): Examples of tag usage  TAGS: []
    tasks:
      task#2    TAGS: [job2]
      task#5    TAGS: []
      task#6    TAGS: [job3]
      task#7    TAGS: [job2, job3]
piyo@vmpc:~/example/tag$

–tags パラメーターと –skip-tags パラメーターを併用¶

-t / --tags パラメーターと --skip-tags パラメータを併用した場合 and 条件になります。

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml -t job2 --skip-tags job1

PLAY [Examples of tag usage] ****************************************************************************************************************

TASK [task#2] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

TASK [task#7] *******************************************************************************************************************************
ok: [hydrogen] => {
    "msg": "Hello world!"
}

PLAY RECAP **********************************************************************************************************************************
hydrogen                   : ok=2    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0   

piyo@vmpc:~/example/tag$ 

piyo@vmpc:~/example/tag$ ansible-playbook -i hosts.yml tag1.yml -t job2 --skip-tags job1 --list-tasks

playbook: tag1.yml

  play #1 (all): Examples of tag usage  TAGS: []
    tasks:
      task#2    TAGS: [job2]
      task#7    TAGS: [job2, job3]
