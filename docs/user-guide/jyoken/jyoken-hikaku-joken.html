比較条件¶

2 つの項目を比較する条件です。

比較演算子¶

比較演算子

true の状態

X == Y

X の値と Y の値が等しいとき

X != Y

X の値と Y の値が等しくないとき

X > Y

X の値が Y の値より大きいとき

X >= Y

X の値が Y の値より大きいか等しいとき

X < Y

X の値が Y の値より小さいとき

X <= Y

X の値と Y の値より小さいか等しいとき

使用例¶

比較条件の使用例です。使用するインベントリです。

---
all:
  hosts:
    centos83:
    centos79:
    ubuntu2004:
    ubuntu1804:
    debian108:
    debian913:
    opensuse152:
    fedora33:

インベントリ内のホスト名は OS のディストリビューション名 + バージョンです。

Debian 系列の管理対象ノードだけ再起動するプレイです。

---
- name: Example usage of the When directive
  hosts: all

  tasks:
    - name: Reboot the managed nodes in the Debian family
      ansible.builtin.reboot:
      when: ansible_facts['os_family'] == "Debian"
      become: yes

実行ログです。 Debian 系列の管理対象ノードだけが再起動しました。
