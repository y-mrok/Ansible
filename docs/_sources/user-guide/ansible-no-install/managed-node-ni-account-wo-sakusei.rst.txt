##################################################
管理対象ノードにアカウントを作成
##################################################
管理対象ノード（ CentOS 8.3 (2011) ）でコントロールノードから接続するときのアカウント ansible-user を作成します。アカウント ansible-user には　sudo 権限を付与します。

.. note::

   この作業はすべての管理対象ノードで実施します。
