.. _filter-list:

##################################################
list 関連のフィルター
##################################################
- リスト（シーケンス）型のデータの中から最大値 / 最小値を返す
- リスト（シーケンス）型のデータの中から最初の値 / 最後の値を返す

.. _filter-list-koubun:

**************************************************
構文
**************************************************
- 最大値

   .. code-block:: yaml

      変数 | max

- 最小値

   .. code-block:: yaml

      変数 | min

- 最初

   .. code-block:: yaml

      変数 | first

- 最後

   .. code-block:: yaml

      変数 | last
