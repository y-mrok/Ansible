##################################################
PATH 関連のフィルター
##################################################
- 変数（ PATH ）からファイル名やディレクトリを抽出する

.. _filter-path-koubun:

**************************************************
構文
**************************************************
- ファイル名を抽出

   .. code-block:: yaml

      変数 | basename

- ディレクトリを抽出

   .. code-block:: yaml

      変数 | dirname
