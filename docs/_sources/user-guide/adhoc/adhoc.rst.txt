##################################################
アドホックコマンド
##################################################
- 「アドホックコマンド」はちょっとした操作をするためにコマンドラインで実行するコマンドのこと
- Ansible はアドホックコマンドで Linux コマンドや Ansible のモジュールを実行する
- Ansible は :command:`ansible` コマンドを使用し、インベントリファイルに記載した管理対象ノード上でアドホックコマンドを実行する
